## 3 : Sea mesh 에 커스텀 shader 입히기 (ShaderMaterial 활용)

밋밋한 단색의 바다를 완전히 다르게 꾸며보는 단계입니다. 그래픽스 지식이 없는 학생들도 프롬프트 몇 줄로 복잡한 쉐이딩(Shading) 코드를 만들어내는 Vibe Coding의 위력을 체감할 수 있습니다.

### "현재 바다 부분을 용암으로 바꾸고 싶은데, 어떤 식으로 코드를 수정하면 좋을지 계획을 먼저 작성해줘."

#### 예상 AI 답변 (계획)
1. **재질(Material) 변경**: 현재 바다는 단순한 색상(`MeshPhongMaterial`)으로 되어 있는데, 이를 직접 프로그래밍할 수 있는 특수 재질(`ShaderMaterial`)로 바꿉니다.
2. **시간 변수 추가**: 용암이 끓어오르는 것처럼 움직이게 하려면 시간이 지남에 따라 변하는 값을 추가해야 합니다.
3. **용암 움직임 만들기**: 파도치는 바다 코드를 수정해서, 버텍스(꼭짓점)들이 불규칙하게 오르락내리락하도록 수학 함수를 적용합니다.
4. **용암 색상 칠하기**: 높이가 높은 곳은 밝은 노란색, 낮은 곳은 어두운 붉은색으로 칠해지도록 색상 그라데이션 코드를 작성합니다.
5. **애니메이션 재생**: 화면이 매 프레임 다시 그려질 때마다 시간 변수를 업데이트하여 용암이 끊임없이 움직이게 만듭니다.

### "계획이 아주 좋아. 이 계획대로 바다를 빨갛게 끓어오르는 용암으로 바꾸는 코드를 작성해서 적용해줘."

*(실제 코드 적용 및 Live Server에서 용암 바다 확인)*

### "용암 효과는 멋진데, 색깔이 조금 단조로워. 가장 높이 솟아오른 부분은 더 밝은 노란색으로 빛나게 하고, 용암이 끓어오르는 속도를 지금보다 1.5배 빠르게 수정해줘."
